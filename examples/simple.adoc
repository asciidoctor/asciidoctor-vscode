= My First AsciiDoc Document
:showtitle:

This is a short introduction to AsciiDoc's core capabilities.
The preview updates as you type, including inline formatting like *bold*, _italic_, and `monospaced` text.

== Headings

Section titles start with one or more `=` signs.
Each extra `=` goes one level deeper, so this section is a level-2 heading.

== Custom styling

You can point the preview at your own CSS stylesheet with the
`asciidoc.preview.style` setting. This folder already does so in
`.vscode/settings.json`, pointing at `css/custom-preview.css` — open the
preview to see the accent color, styled headings, and tinted code blocks.
The preview refreshes automatically, just as you type.
