= Quick Start
:description: Open your first AsciiDoc file and preview it in seconds.

The extension activates automatically when you open an AsciiDoc file (`.adoc`, `.ad`, `.asc`, `.asciidoc`).

== Your first preview

. Open an AsciiDoc file, or create a new one and save it with a `.adoc` extension.
. Press kbd:[Ctrl+Shift+V] (Mac: kbd:[Cmd+Shift+V]) to toggle the preview, or kbd:[Ctrl+K] kbd:[V] (Mac: kbd:[Cmd+K] kbd:[V]) to open it side by side.
. Start typing. The preview updates as you write.
