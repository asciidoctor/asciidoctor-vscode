= Contributing
:description: How to contribute to the AsciiDoc extension.

Contributions are welcome.
Clone the repository, commit your changes, and open a pull request.
In the pull request description, clearly highlight what you changed.

[NOTE]
====
Do not update the extension version -- maintainers do this when a new version is released.
====

== Documentation changes

You are free to fix typos and errors, and to add or improve descriptions.
If you have a larger style change in mind, please open an issue (or a dedicated pull request) first so it can be discussed.
